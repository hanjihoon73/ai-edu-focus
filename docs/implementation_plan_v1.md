# Article-1~4 웹페이지 공유 및 배포 계획

제이슨, 현재 작성된 기사들을 다른 사람들에게 멋지게 공유하려면, 각 파일들을 하나하나 보내는 것보다 **하나의 웹사이트(포털)**처럼 구성해서 URL 링크 하나로 공유하는 것이 가장 좋습니다.

이를 위해 메인 페이지(`index.html`)를 만들고, 가장 쉽고 빠른 배포 방법인 **Vercel** 또는 **GitHub Pages**를 활용하는 것을 제안합니다.

## 안내 사항

> [!IMPORTANT]
> 1. 현재 `article-3`, `article-4` 파일이 없습니다. 제가 기존 스타일을 유지하며 템플릿을 미리 만들어둘까요?
> 2. 배포 방식 중 어떤 것을 선호하시나요? (Vercel: 가장 빠름 / GitHub Pages: 저장소 연동 편리)

## 변경 사항 및 추진 계획

기사들을 한눈에 볼 수 있는 'AI에듀:포커스' 포털 페이지를 추가하고, 전체 구조를 정리합니다.

### 1. 메인 포털 및 구조 정리

- **[NEW] index.html**: 기사 4개를 카드 형태로 보여주는 메인 랜딩 페이지
- **[NEW] article-3-1.html, article-4-1.html**: 누락된 기사를 위한 기본 템플릿
- **[MODIFY] styles.css**: 인덱스 페이지를 위한 스타일링 추가

### 2. 폴더 구조 (예정)

```text
ai-edu-focus/
├── index.html (포털 메인)
├── article-1-2.html
├── article-2-2.html
├── article-3-1.html
├── article-4-1.html
├── styles.css
├── img/
└── docs/ (문서 관리)
```
